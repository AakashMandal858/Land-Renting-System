# Land Renting System

## Description
Command-line Python application to manage land renting and returns. Stores all data in a file system with modular operation handling.

## Features
- Rent and return lands
- Billing management
- File-based data storage
- Easy to use and extend

## Technologies
- Python
- File handling
- datetime module

## Usage
- Run `main.py` to start the program
- Follow on-screen prompts for operations
